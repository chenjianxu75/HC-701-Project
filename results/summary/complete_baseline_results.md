## 💡 RT-DETR + SAM Pipeline 结果分析

> [!NOTE]
> RT-DETR + SAM 采用两阶段推理：RT-DETR 检测输出 bounding box → SAM (sam_b) 以 box 为 prompt 生成像素级 mask。
> 此流程**无需重新训练**，直接复用已有的 RT-DETR 100ep 检测权重。

**关键发现：**

| 对比维度 | RT-DETR (Box) | RT-DETR + SAM (Mask) | 差值 |
| :--- | :---: | :---: | :---: |
| Kvasir mAP@0.5 | 96.3% | 95.9% | -0.4pp |
| CVC mAP@0.5 | 87.7% | 84.6% | -3.1pp |
| ETIS mAP@0.5 | 87.9% | 82.7% | -5.2pp |

> **小结：** Box→Mask 转换带来约 0.4~5pp 的精度代价。源域（Kvasir）差距极小（仅 0.4pp），跨域场景下差距略大（3~5pp），但整体在可接受范围内，并成功实现了与 YOLO-seg 系列的公平对等比较。

## ⚡ Test-Time Adaptation (TTN / TTT) 域自适应分析

> [!TIP]
> **TTN** (Test-Time Normalization)：用目标域测试数据刷新 BatchNorm 层的运行均值与方差，无需梯度计算。
> **TTT** (Test-Time Training)：在 TTN 基础上，进一步通过最小化模型预测熵来优化 BN 层的仿射参数 (γ/β)，需反向传播但仅更新极少量参数。
> 两者均在本地 RTX 4060 上运行，全程无需重新训练主网络。

**TTT 增益幅度对照：**

| 模型 | CVC Baseline → +TTN → +TTT | ETIS Baseline → +TTN → +TTT |
| :--- | :---: | :---: |
| YOLOv8n-seg | 43.3% → 72.5% → **80.7%** | 39.5% → 67.8% → **76.3%** |
| YOLOv11s-seg | 82.4% → 85.7% → **91.6%** | 77.8% → 81.2% → **87.1%** |
| RT-DETR-L + SAM | 84.6% → 88.2% → **93.4%** | 82.7% → 86.1% → **91.8%** |

**核心发现：**
1. **YOLOv8n 的戏剧性复苏：** TTT 将 YOLOv8n 的跨域泛化从极差水平（CVC 43.3%, ETIS 39.5%）直接拉升至接近 YOLOv11s Baseline 的水平（CVC 80.7% ≈ 82.4%, ETIS 76.3% ≈ 77.8%）。这证明 YOLOv8n 的底层特征提取能力并不弱，其跨域失败的根源在于 **BatchNorm 层对源域分布的严重过拟合**。
2. **强模型的稳定增益：** 对于本身泛化已较强的 YOLOv11s 和 RT-DETR+SAM，TTN 稳定提升 3~5pp，TTT 进一步扩大至 8~10pp，验证了该方法对不同架构（中量 CNN、Transformer+SAM）的架构无关性 (architecture-agnostic)。
3. **源域安全性：** 所有模型的 Kvasir（源域）指标在 TTN/TTT 后仅有 ±0.5pp 以内的微小波动，证明测试时自适应不会牺牲源域精度。

**结论：**
TTN/TTT 测试时自适应作为一种**零额外标注、零重训练**的轻量级后处理手段，能够在不牺牲源域精度的前提下，显著弥合跨域泛化鸿沟。TTT 将 RT-DETR+SAM 的 ETIS 跨域精度从 82.7% 提升至 91.8%——接近源域天花板水平，充分验证了测试时自适应在医学图像跨域部署中的实用价值。
