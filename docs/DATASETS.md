# Datasets

This repository does not commit the raw datasets. Download them separately, place them in the paths below, and then run the conversion scripts from the repository root.

## Official Sources

- Kvasir-SEG (official): [datasets.simula.no/kvasir-seg](https://datasets.simula.no/kvasir-seg/)
- CVC-ClinicDB (official challenge page): [polyp.grand-challenge.org/CVCClinicDB](https://polyp.grand-challenge.org/CVCClinicDB/)
- ETIS-LaribPolypDB (official challenge page): [polyp.grand-challenge.org/EtisLarib](https://polyp.grand-challenge.org/EtisLarib/)

## Accessible Mirrors

- CVC-ClinicDB mirror: [Kaggle CVC-ClinicDB](https://www.kaggle.com/datasets/balraj98/cvcclinicdb)
- ETIS-LaribPolypDB mirror: [Kaggle ETIS-LaribPolypDB](https://www.kaggle.com/datasets/nguyenvoquocduong/etis-laribpolypdb)

## Conversion Commands

Generate the YOLO segmentation datasets:

```bash
python scripts/data/convert_kvasir.py
python scripts/data/convert_cvc.py
python scripts/data/convert_etis.py
```

Generate the RT-DETR detection versions:

```bash
python scripts/data/prepare_rtdetr_data.py --dataset all --overwrite
```

## Notes

- CVC-ClinicDB and ETIS-Larib are used here as external test domains in the main Kvasir-trained setup.
- The repository keeps only the scripts and documentation. Raw data stays ignored by `.gitignore`.
